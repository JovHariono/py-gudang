Aplikasi Python untuk stok gudang

**Features**
- Add new items
- View all items
- View item details
- Update existing items
- Delete items
- Input validation
- Item code format validation using Regular Expression

**Run Application**
python warehouse_inventory.py

Main Menu
=== SISTEM GUDANG ===

1. Tambah Barang
2. Lihat Barang
3. Update Barang
4. Hapus Barang
5. Keluar

**Example Usage**
**Add Item**
Masukkan kode barang: A-001
Masukkan nama barang: Keyboard
Masukkan jumlah barang: 10

**View Item**
Kode: A-001
Nama: Keyboard
Jumlah: 10

**Update Item**
Masukkan kode barang yang ingin diupdate: A-001
Masukkan nama barang baru: Mechanical Keyboard
Masukkan jumlah barang baru: 20

**Delete Item**
Masukkan kode barang yang ingin dihapus: A-001
